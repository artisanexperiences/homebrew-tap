# Artisan Experiences Homebrew Tap

This repository contains Homebrew formulas for [Artisan Experiences](https://github.com/artisanexperiences) projects.

## Installation

```bash
brew tap artisanexperiences/tap
brew install arbor
```

## Available Formulas

### Arbor

Git worktree manager for Laravel projects.

**Install:**
```bash
brew install arbor
```

**Upgrade:**
```bash
brew upgrade arbor
```

**Documentation:**
- [Arbor GitHub Repository](https://github.com/artisanexperiences/arbor)
- [Release Notes](https://github.com/artisanexperiences/arbor/releases)

## About

This tap is maintained by the Artisan Experiences team. Formulas are automatically updated when new releases are published.
